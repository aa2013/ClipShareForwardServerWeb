该项目是 [ClipShare](https://github.com/aa2013/ClipShare) 的中转程序的后台管理页面。

该管理页面主要提供以下功能：

+ 简单的网络速度/流量/连接数的实时查看
+ `config.yaml` 配置文件修改
+ 私有模式下的`套餐` 和 `密钥` 的管理
+ 运行日志查看
